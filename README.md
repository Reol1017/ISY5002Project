# ISY5002Project

# For Windows
**Start Backend:**  
  Enter the backend path:  
  **cd ISY5002Project/backend**  
  Install requirement:  
  **pip install Flask, CORS, requests, pandas, datetime**  
  Switch on Application:  
  **python app.py**  

Start Frontend: (vue + element plus)  
  Enter the frontend path:  
  **cd ISY5002Project/frontend**  
  Install requirement:  
  **npm install**  
  Switch on Application:  
  **npm run dev**
